## Kubernetes Cluster

### MiniKube for Ubuntu

```
minikube start --kubernetes-version v1.11.5

minikube ssh

minikube dashboard

minikube ip

minikube logs

minikube docker-env
```

### Docker for Mac

### Kubectl Configuration

```
kubectl config view
```

### Cluster information

```
kubectl cluster-info
```
### Node information

```
kubectl get nodes

kubectl get nodes -owide

kubectl describe nodes
```
